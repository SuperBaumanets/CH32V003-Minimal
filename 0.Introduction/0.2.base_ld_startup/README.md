## Проверка размещения секций в памяти CH32V003

README для отладки скрипта линкера (`ld`) и настройки startup файла. Необходим, чтобы убедиться, что при старте копируется содержимое **.data** секции из FLASH в SRAM, зануляется секция **.bss** и верно устанавливаются **gp** и **sp**.

### Сборка прошивки
```bash
# Базовая сборка
make all
```

После успешной сборки в папке `build/` будут созданы файлы:
- `firmware.elf` - ELF файл для отладки
- `firmware.elf.dump` - Дамп ELF файл для отладки
- `firmware.bin` - бинарный файл для прошивки
- `firmware.map` - MAP файл для отладки
